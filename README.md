# recruitment-task
Html,js,css project.

## Installation

1. Clone this repo

```sh
git clone https://github.com/1991Pawel/recruitment-task.git
```

2. Install NPM packages

```sh
npm install 
```
3. Start a development server
npm 

```sh
npm run dev
```
